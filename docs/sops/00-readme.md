# SOPs — Standard Operating Procedures

Esta carpeta contiene los procesos operativos del negocio. Es la fuente de verdad de cómo trabajamos.

## Para qué sirve

- **Onboarding** de personas nuevas (Adrián, equipo futuro)
- **Documentación** de procesos para que perduren
- **Knowledge source** del agente IA (yo los leo siempre que arranque una conversación)
- **Coherencia** del equipo en cómo accionamos

## Cómo se gestionan

- Cada SOP es un archivo `.md` en `docs/sops/`
- Numerados con prefijo (`01-`, `02-`...) para orden
- Sin jerarquía profunda al inicio: lista plana
- Versionados con git
- Visibles desde el OS en `/sops`

## SOPs disponibles

Lista la genera la página `/sops` automáticamente leyendo todos los `.md` de esta carpeta.
