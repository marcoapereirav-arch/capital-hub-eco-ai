## LA SEMANA DE UN VISTAZO

```
┌─────────┬────────────────────────────────┬─────────────────┐
│ DÍA     │ ACCIÓN                         │ PUBLICACIÓN     │
├─────────┼────────────────────────────────┼─────────────────┤
│ LUN AM  │ Sync · Analizar · 5 Guiones    │ 13h · Viral 1   │
│ MAR PM  │ Grabar los 5 en batch          │ 20h · Conexión  │
│ MIÉ     │ Editar · Marcar "Grabados"     │ 09h · Viral 2   │
│ JUE     │ Responder comentarios (30 min) │ 20h · CTA       │
│ VIE     │ Retrospectiva · Qué funcionó   │ 12h · Viral 3   │
│ SÁB     │ OFF                            │ 12h · (auto)    │
│ DOM     │ Prep siguiente semana (10min)  │ 11h · Conexión  │
└─────────┴────────────────────────────────┴─────────────────┘
```

Total horas activas/semana: **~6h** · Output: **5 videos publicados**

---

## LA FÓRMULA 4+1

```
3 VIRALES (sin CTA)    →  CREZCO
1 CTA (lead magnet)    →  CONVIERTO
1 CONEXIÓN (personal)  →  CONFÍAN
```

---

## LAS 5 REGLAS NO NEGOCIABLES

```
1. SOLO grabo en BATCH. Nunca día a día.
2. El estado (Pendiente → Grabado → Publicado) SIEMPRE se marca.
3. 5 videos/semana. Ni más, ni menos.
4. Máx 3 tomas por video. Si no sale, siguiente.
5. Comentarios: o 30 min bien, o nada.
```

# SI DUDAS, MIRA ESTO

**¿No sé qué grabar?** → Webapp · Mis guiones · Pendientes.

**¿No tengo guiones pendientes?** → Vuelvo al LUNES y genero 5.

**¿El ángulo no me cuadra?** → Chat del guion · "cámbialo así".

**¿No responde mi audiencia?** → Retrospectiva VIERNES · qué patrón cambio.

**¿Estoy cansado?** → Ajusto día, no elimino sesión. La máquina se adapta a mí.
