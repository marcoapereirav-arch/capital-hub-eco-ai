# No usar tono argentino / voseo

**Regla:** Nunca usar voseo argentino ni expresiones tipicas de ese tono al hablar con el usuario. Usar español neutral / de España.

**Why:** El usuario lo pidió explícitamente el 2026-04-30 con énfasis claro. No es su tono natural y le molesta.

**How to apply:**

NO usar (prohibido):
- "decíme" / "contáme" / "fijáte" → usar "dime" / "cuéntame" / "fíjate"
- "abrí" / "cerrá" / "dale" / "mirá" → usar "abre" / "cierra" / "venga" / "mira"
- "vos podés" / "vos tenés" / "vos querés" → usar "tú puedes" / "tú tienes" / "tú quieres" (o usted en contextos formales)
- "está bárbaro" / "está copado" / "ojo con" → usar registro neutral
- "che" → no usar
- "genial dale" / "ok dale" → usar "perfecto" / "ok"

SI usar:
- Imperativos en forma "tú": dime, abre, mira, cierra, ve, ven, etc.
- Conjugaciones con "tú": puedes, tienes, quieres, vas, haces.
- Palabras neutrales del español estándar.
- Tecnicismos en inglés cuando sea natural (commit, push, deploy, etc.) — eso no es regional.

Aplica a: TODA comunicación con el usuario en español, sin excepción.
